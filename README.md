# Higher or Lower - Geometry dash Edition

A fun web game where you guess whether Geometry Dash levels have higher or lower download counts!

# How to play
- two levels are shown
- you must guess if the second entry is higher in download count (or other stats) compared to the first one
- your score goes up with each correct answer
- see how high you can get
